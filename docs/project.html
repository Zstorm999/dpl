Duck Programming Language (DPL) is a project that aims at creating a programming language that fulfils the needs of modern programmers while still being simple to learn and easy to read. Its main goal is to find balance between readability and performance, and being a modern and competitive alternative to somewhat harder to learn languages like C++.

Here are presented the different versions, aka the plan, that the project will follow during its conception. Each step will wait for the previous to be completed, and a the end of each step a new version will be released, that you can find on the downloads page.
Beware that some things might change, this list is not at all complete.

  • V0.1 : Integer arithmetic

    • int data type
    • char data type
    • bool data type
    • assignment operator: =
    • basic arithmetic operators: + - * / %
    • control flow structures: if, else and for
    • basic implementation of print

  • V0.2 : Functions and pointers

  • V0.3 : Floating point arithmetic

  • V1.0 : Official release

    • compiler for all major OS: Linux, Windows, macOS
    • support for CUDA, and other GPU computing tools

  • Cross language capabilities

    • C/C++
    • Go

This isn't the end of DPL! We still have a ton of ideas that might be added later, so check this website and our social networks regularly to stay informed of upcoming updates and events.
